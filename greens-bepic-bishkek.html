Состав Gr8 Greens

Перечень ингредиентов

Смесь биологически активных трав (1800 мг на порцию )

Содержит сухие экстракты сока таких злаковых трав:

  • Ячмень. Способствуют лучшему пищеварению, повышает уровень энергии, регулирует уровень сахар в крови.
  • Люцерна. Помогает в детоксикации организма, борется с воспалением, ускоряет восстановление после физической активности, регулирует уровень сахар в крови.
  • Овёс. Помогает в борьбе со свободными радикалами, имеет успокаивающее воздействие на стенки желудка
  • Пшеница. Выполняет детоксикацию организма, подавляет воспаление, стимулирует иммунную систему, ускоряет восстановление после физической активности.
  • Вайтграсс (молодые ростки пшеницы). Поддерживает иммунную систему, защищает организм от свободных радикалов. помогает снизить уровень холестерина и укрепляет сердечно-сосудистую систему.
  • Камут (дикая пшеница) . Обеспечивает организм энергией, снижают уровня холестерина, имеет благоприятное воздействие на здоровье сердца, является антиоксидантом, поддерживает здоровье пищеварительной системы.

Смесь для улучшения выработки оксида азота (4858 мг на порцию )

  • L-аргинин (условно-незаменимая аминокислота)
  • L-цитруллин (натуральная α-аминокислота)
  • Агматин сульфат (производное аминокислоты аргинина )
  • Ацетилцистеин   (производное аминокислоты цистеина )
  • Экстракт свеклы
  • Ресвератрол (извлекается из плодов винограда)

Смесь для регулирования чувствительности к инсулину (285 мг на порцию)

  • OmniMinAC™ (стандартизованный комплекс антиоксидант, микро- и макроэлементов)
  • Экстракт мумиё (органо-минеральный продукт природного  происхождения). Ускоряет заживление трофических язв, улучшает пособность клеток организма эффективно использовать инсулин для регуляции уровня сахара.
  • Экстракт банабы (Lagerstroemia speciosa ; растущее в Индии дерево). Банаба содержит коросоловую кислоту и лагерстрозид, которые способствовуют снижению уровня глюкозы в крови и улучшению чувствительности к инсулину.
